软件介绍
✅ 高效处理，省时省力

支持阅读 单页PDF、多页PDF、OFD、JPG、PNG、BMP等混合格式文件，单次处理数量无上限！
支持阅读 增值税普专票、铁路票、航班票、统一收据、银行回单等单据。识别精准度高达98%，关键信息准确读取并分类！
支持 读取单票据多行（条）信息

支持 对识别的票据文件进行结构化重命名，无需打开文件即可快速了解文件内核心信息！
支持 多文件合并、智能分类归档功能！
